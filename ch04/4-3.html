4-3. 콜백 함수는 함다

콜백 함수는 함수다

  • 어떤 객체의 메서드를 전달하더라도 메서드가 아닌 함수로서 호출. ⇒ this 는 메서드가 속한 객체가 아닌, 전역 객체 혹은 콜백 함수를 호출할때 바인딩한 this 를 따라감

    var obj = {
  vals: [1, 2, 3],
  logValues: function (v, i) {
    console.log(this, v, i);
  }
};

obj.logValues(1,2); // obj
[4, 5, 6].forEach(obj.logValues); // Window \n Window \n Window
